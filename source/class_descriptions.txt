BarycentricFrame: Класс координат относительно барицентра

CartesianFrame: Класс декартовых координат относительно центра Земли

CelestialCoord: Класс небесной координаты - часы, минуты, секунды

Converter: Класс для переводов между системами и манипуляциями в них
    - Перевод из цилиндрической системы координат в декартову
    - Задание даты в TT на основе даты юлианской системы
    - Перевод в TBD
    - Интерполяция положения Хаббла
    - Перевод из земных координат в геоцентрические небесные
    - Интерполяция центра Земли
    - Интерполяция координат центра Земли для численного интегрирования
    - Перевод сферических координат из "часы-минуты-секунды" в радианы
    - Перевод барицентрических координат в сферические
    - Интерполяция для выявления смоделированных значений с нужными датами
    - Поправка за световой промежуток
    - Нахождение координат по измененной дате
    - Поправка на гравитационное отклонение света
    - Аберрация

CylindricalFrame: Класс цилиндрических координат



DataHandler: Класс, читающий данные из файлов
    - Считывание данных наблюдений
    - Считывание данных местоположений обсерваторий
    - Считывание данных для интерполяции положения Хаббла
    - Считывание данных для интерполяции времени
    - Считывание данных для интерполяции центра небесного тела

Date: Класс, создающий дату из строки в формате "год месяц день.доля дня" и хранящий представление даты в разных системах

Facade: Класс, вызывающий методы других классов
    - Считывание данных
    - Перевод времени наблюдений
    - Численное интегрирование
    - МНК (пока в процессе)
    - Запись полученных модельных данных в файл

GeocentricFrame: Класс геоцентрических координат

IntegrationVector: Вектор для численного интегрирования

InterpolationHubbleFrame: Класс для хранения данных интерполяции Хаббла

InterpolationTimeFrame: Класс для хранения данных для интерполяции времени

LeastSquares: МНК

Observation:  Класс для хранения наблюдения

ObservatoryData: Класс для хранения данных обсерватории

OrbitalIntegration: Класс для численного интегрирования
    - Метода Дормана-Принса

SphericalFrame: Класс для сферических координат (в радианах)

Velocity: Класс для хранения скорости
